# Trading competition repository
Contributors: Liam, Oliver, Albert, Pliego, Kanish, Simon

Sanitised copy of a private repository as a display of work for trading algorithms.

My primary contributions are in algorithms for trading volatilities as well as predictive price impacts from large positions on books. Includes optimisation algorithms for how to best spread trades for large positions, and a self-designed optimisation function for that uses a predicition of rebalancing costs when delta hedging straddles within limits.
